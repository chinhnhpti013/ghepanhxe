📋 Tổng quan

Công cụ tự động ghép ảnh hiện trường xe ô tô thành file PDF khổ A4 chuẩn với tiêu đề, logo PTI SOS, caption từng hạng mục và số trang. Quy trình gồm 2 bước: đặt tên file theo thư mục → ghép ảnh vào khung PDF.

Python 3.10+ Pillow ≥ 10 ReportLab ≥ 4 openpyxl ≥ 3.1 python-docx ≥ 1.1
📁

1

Giai đoạn 1 — Đặt tên file

dat_ten_file.py — Đọc thư mục con, sao chép và đổi tên ảnh
ℹ️
Thư mục renamed/ sẽ tự động xóa sạch và tạo lại mỗi lần chạy — tránh file cũ gây lỗi.
Terminal
python dat_ten_file.py

Kết quả: ảnh được sao chép sang renamed/ với tên dạng 1. Đèn hậu bên phải vỡ_1.jpg, ..._2.jpg và file mapping.json lưu ánh xạ tên → caption.

2

Giai đoạn 2 — Ghép ảnh thành PDF

ghep_anh.py — Tạo file PDF khổ A4 từ ảnh đã đặt tên

Dọc (Portrait A4)

Grid 2×3 — 6 ảnh/trang
--layout doc

Ngang (Landscape A4)

Grid 2×2 — 4 ảnh/trang
--layout ngang

Lệnh cơ bản
python ghep_anh.py --gdv "Nguyễn Văn A" --layout doc
Có ngày giám định
python ghep_anh.py --gdv "Trần Thị B" --layout doc --ngay "14/06/2026"
Truyền BKS và Ga-ra thủ công
python ghep_anh.py --gdv "Lê Văn C" --layout ngang --bks "14C-402.84" --gara "Gara XYZ"
Tham số Mô tả
--gdv Bắt buộc Tên giám định viên, ví dụ: "Nguyễn Văn A"
--layout doc (mặc định) hoặc ngang
--ngay Tùy chọn Ngày giám định, ví dụ: "14/06/2026"
--bks Tùy chọn Biển kiểm soát (nếu không có Bao_gia.xlsx)
--gara Tùy chọn Tên ga-ra / địa điểm
--output Tùy chọn Đường dẫn file PDF đầu ra tùy chỉnh
